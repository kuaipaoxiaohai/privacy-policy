# privacy-policy

This application is committed to protecting user privacy. This policy outlines our practices regarding information handling.

1.**Data Collection and Use** 
The App is a standalone application that operates locally. We do not collect, store, or transmit any personal identifiable information (PII), device information, or geolocation data.

2.**Data Storage** 
All data generated during use (such as learning progress or notes) is stored exclusively on the user's local device. We have no access to, and cannot extract or back up, this data.

3.**Third-Party Services** 
The App does not integrate any third-party SDKs, advertising plugins, or analytics tools. Consequently, no third-party entities collect your information through this App.

4.**Permissions** 
The App only requests system permissions strictly necessary for its core functions. We do not access sensitive permissions (e.g., Camera, Photos, Contacts) without your explicit authorization for a specific feature.
